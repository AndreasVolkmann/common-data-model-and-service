# Data Format

```

{

    Int16,
    Int32,
    Int64,
    Float,
    Double,
    Guid,
    String,
    Char,
    Byte,
    Binary,
    Time,
    Date,
    DateTime,
    DateTimeOffset,
    Boolean,
    Decimal,
    Json
}
```
