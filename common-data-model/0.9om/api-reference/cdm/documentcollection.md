# Document Collection

A document collection extends [Collection](collection.md) and adds additional behaviors specific to document collections.

```
public class CdmDocumentCollection extends CdmCollection<CdmDocumentDefinition>
```

## Constructors
|Name|Description|
|---|---|
|**CdmDocumentCollection(CdmCorpusContext, [CdmFolderDefinition](folder.md))**<br/>*ctx*: The corpus context.<br/>*owner*: The folder that contains this collection.|Initializes a new instance of the CdmDocumentCollection class.|
